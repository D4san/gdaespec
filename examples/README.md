# Examples

This folder contains short notebooks that show how to apply the trained models
without walking through the full experiment workflow.

| Notebook | Purpose |
| --- | --- |
| `Quick_Start_GDAE_Workflow.ipynb` | Earth-like G-DAE example using one campaign observation and its clean reference. |

The input files are `data/observation.dat` and `data/reference.dat`. Both are
small copies derived from the Earth-like retrieval campaign.
