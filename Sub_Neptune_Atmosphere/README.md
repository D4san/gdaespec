# G-DAE Training on Sub-Neptunes (K2-18b Analog)

This directory contains the K2-18b/Sub-Neptune experiment. It is organized as a
numbered notebook workflow: generate spectra, compute stellar contamination,
train the autoencoder, and evaluate the reconstruction.

## Workflow

Run or read the notebooks in this order:

1. `01_Spectra_Generation.ipynb`
   Generates the synthetic spectral library for the K2-18b analogue.
2. `02_Stellar_Contamination.ipynb`
   Computes the stellar contamination factor `epsilon(lambda)` for the selected
   spot and facula configurations.
3. `03_AE_Training.ipynb`
   Trains the G-DAE model using the generated spectra and contamination
   profiles.
4. `04_G-DAE_Evaluation.ipynb`
   Evaluates the trained model and compares contaminated, clean, and
   reconstructed spectra.

## Directory Contents

| Path | Purpose |
| --- | --- |
| `specs/` | Generated or stored K2-18b spectral datasets. |
| `TLS/` | Stellar contamination curves used by the Sub-Neptune workflow. |
| `Phoenix/` | Stellar spectra downloaded by the spectra-generation workflow. |
| `CIA/` | Collision-induced absorption inputs downloaded by the spectra-generation workflow. |
| `opacidades-todas/` | Local opacity tables downloaded by the spectra-generation workflow. |
| `plots/` | Figures generated by the analysis notebooks. |
| `AE.keras` | Final trained model produced by `03_AE_Training.ipynb` and used by the evaluation notebook. |
| `waves.txt` | Wavelength grid used by this experiment. |

## Practical Notes

- `01_Spectra_Generation.ipynb` downloads or prepares the local `CIA/`,
  `Phoenix/`, and `opacidades-todas/` inputs used to generate the spectra.
  These folders are not intended for GitHub because of their size.
- `AE.keras` is the final trained model generated by `03_AE_Training.ipynb`.
- The folder is self-contained for the K2-18b case; the Earth-like retrieval
  experiments live separately in `../Earth_like_Atmosphere/`.
