# SPHINX Stellar Grids

This folder contains the SPHINX stellar grids used by
[../generate_sphinx_trappist1_contamination.ipynb](../generate_sphinx_trappist1_contamination.ipynb).

The `Teff_*` files are input grids. The generated contamination factors
(`sphinx_TRAPPIST-1_contam_*`) are written to the parent
`stellar_contamination/` folder so they sit next to the PHOENIX-based
contamination products.
